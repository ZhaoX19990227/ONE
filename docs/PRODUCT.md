# ONE MVP 产品范围

## 产品定位

ONE 不是传统陌生人社交产品，而是以“共同完成一件事”为核心的活动匹配平台。首发建议聚焦一个城市的下班后轻活动，并从羽毛球、桌游等高结构化品类开始运营。

## 北极星指标

每周真实完成的活动席位数。

辅助指标包括活动成局率、报名到场率、发布到首次报名时长、30 天复参与率、每千次参与的有效举报数。

## 核心闭环

1. 用户选择兴趣、城市、常用时间和活动边界。
2. 首页根据时间、距离、水平和可靠性推荐活动。
3. 发起人通过模板结构化发布活动。
4. 用户报名，活动支持自动通过、人工审核和候补。
5. 报名成功后在活动房间接收公告和结构化问答。
6. 用户签到并完成活动。
7. 双向评价沉淀完成次数、取消和爽约事实。
8. 系统推荐复约、周期活动和相邻兴趣。

## P0 功能

- 微信登录与最小化资料
- 推荐首页、搜索和筛选
- 活动模板、发布、详情、取消
- 报名、审核、候补、名额并发控制
- 我的行程、活动公告、订阅提醒入口
- 签到、完成确认、评价、举报、拉黑
- 运营后台的活动、用户和举报审核

## 约束

- 首发仅限 18 岁以上用户。
- 精确地址仅对报名成功用户显示。
- 普通微信群只是可选沟通层，业务状态以 ONE 为准。
- MVP 不提供自由私聊和平台代收代付。
- “鸽子金”在 UI 中是履约约束概念；真实资金能力上线前必须完成支付、退款、投诉和合规确认。
